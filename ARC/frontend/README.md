# Bootie Hunter Frontend

Flutter frontend for R.E.E.D. Bootie Hunter application.

## Setup

1. Install Flutter dependencies:
```bash
flutter pub get
```

2. Run the app:
```bash
flutter run
```

## Project Structure

- `lib/main.dart` - App entry point
- `lib/screens/` - Screen widgets
- `lib/widgets/` - Reusable widgets
- `lib/services/` - API and business logic services
- `lib/providers/` - State management (Provider)
- `lib/models/` - Data models

## Configuration

Update the API base URL in `lib/services/api_service.dart` or use environment variables.
